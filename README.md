# final_project-wuzheyu913

找出n^n的加總後的小數點前十位數字, n=1,2,...,N

說明:
此function為找出 1^1 + 2^2+ 3^3 + ... + N^N 的小數點前十位數字

    INPUT: find_TopTenDigits(N)  % N為一個正數且使用者自己決定
    OUTPUT: 1^1 + 2^2+ 3^3 + ... + N^N 的小數點前十位數字
    
Example:

    INPUT: find_TopTenDigits(10)
    OUTPUT: 
    前十位數字 = 
     0     4     0     5     0     7     1     3     1     7
